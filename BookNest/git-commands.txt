
# Git commands to commit login functionality

# Add all changes
git add .

# Commit with a message
git commit -m "Implement login functionality with JWT authentication

- Added JWT dependencies to pom.xml
- Created LoginRequest DTO
- Created JwtUtil for token generation
- Updated UserService with login method (admin + student authentication)
- Updated AuthController with /login endpoint
- Updated SecurityConfig for auth endpoints
- Updated frontend auth.ts with token/user management
- Updated login.ts to call backend API and redirect by role"

# Push to remote
git push origin main

# git checkout -b blackboxai/login-feature
